``````````````````````````````````````````````````
定时器资源：
TIM12：蜂鸣器-PWM
TIM1：-
TIM3：-
TIM4: -
TIM5：舵机-PWM
TIM6：-

串口资源：
USART1：DBUS
USART2：Debug
USART3：-
USART6：-

IO资源：
I0\H12\H11：光电开关
H10：舵机
D15：摄像头通道选择

电机资源：
底盘：3508*4 + 3508*2 //底盘 + 提升
上层：(3508*2 + 2006*2) + 2006*1 + 3508*1 //旋转、平移 + 补弹 + 推送

``````````````````````````````````````````````````
遥控器功能：
固定功能：右拨杆控制底盘xy 拨轮控制底盘z

左上+右上：小资源岛 左上+右中：大资源岛 左上+右下：兑换区
左中：
左下+右下：补弹电机复位

键盘功能：
WASD+鼠标左右：底盘控制	
R+鼠标左键单次：补弹
F：救援

Q：抓取矿石
E：紧固矿石
G：推送矿石
V：切换视角
C：自动切换提升和推送
B + CTRL：正转/反转抓取
Z/X + CTRL/SHIFT：提升/推送的减少/增加长度

``````````````````````````````````````````````````
抓取任务：（上下）提升、（前后）推送、（左右）平移、旋转
以自身车为方向，
平移向左正，
旋转向内正，
推送向外正，
提升向上正，//OK

``````````````````````````````````````````````````
拖链走线：光电开关*3 + (舵机*3 + 摄像头通道信号*1) + 电磁阀*2
